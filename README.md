## 📋 Descripción

Sitio web frontend para una guía de hoteles que permite a los usuarios:
- Ver información de diferentes hoteles
- Consultar precios y servicios
- Contactar con la empresa
- Navegar por términos y condiciones

## 🛠️ Tecnologías Utilizadas

| Tecnología | Descripción |
|------------|-------------|
| **HTML5** | Estructura semántica del sitio |
| **CSS3** | Estilos personalizados y diseño responsive |
| **JavaScript** | Interactividad y funcionalidades |
| **Gulp** | Automatización de tareas (minificación, optimización) |
| **Bootstrap** | Framework CSS para diseño responsivo |
